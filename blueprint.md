# 로또 번호 추첨기 프로젝트

## 개요

이 프로젝트는 모던하고 세련된 디자인의 로또 번호 추첨 웹 애플리케이션을 제작하는 것을 목표로 합니다. 사용자는 버튼 클릭 한 번으로 쉽고 빠르게 6개의 로또 번호를 생성하고 결과를 확인할 수 있습니다. 가인지컨설팅그룹의 웹사이트에서 영감을 받은 디자인을 적용하여 사용자에게 전문적이고 신뢰감 있는 경험을 제공합니다.

## 디자인 및 기능

### 스타일 및 디자인 (가인지컨설팅그룹 웹사이트 참고)

*   **색상 팔레트:**
    *   주요 색상: 네이비 블루 (`#1A237E`)
    *   보조 색상: 화이트 (`#FFFFFF`), 라이트 그레이 (`#F5F5F5`)
    *   강조 색상: 골드 (`#FFD700`)
*   **타이포그래피:**
    *   헤더 및 주요 텍스트: 'Noto Sans KR' 또는 시스템 기본 sans-serif 폰트
    *   본문: 깔끔하고 가독성 좋은 폰트
*   **레이아웃:**
    *   전체적으로 중앙 정렬된 깔끔한 레이아웃
    *   모바일 반응형 디자인
*   **시각적 요소:**
    *   은은한 배경 텍스처 또는 그라디언트
    *   입체감을 주는 부드러운 그림자 효과
    *   상호작용 요소(버튼 등)에 'glow' 효과 적용

### 주요 기능

*   **로또 번호 생성:**
    *   '번호 생성' 버튼 클릭 시 1부터 45까지의 숫자 중 겹치지 않는 6개의 숫자를 무작위로 추출합니다.
*   **결과 표시:**
    *   생성된 6개의 번호를 시각적으로 강조하여 표시합니다.
    *   각 번호는 개별적인 스타일(예: 원형 배경)을 가집니다.
    *   결과가 표시될 때 부드러운 애니메이션 효과를 적용합니다.
*   **웹 컴포넌트 활용:**
    *   로또 번호를 표시하는 부분을 재사용 가능한 웹 컴포넌트(`lotto-ball`)로 만들어 코드의 모듈성과 유지보수성을 높입니다.
